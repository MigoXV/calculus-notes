## 2.向量积的定义

设 $\vec{a}, \vec{b}$ 的夹角为 $\theta$ ，定义
向量 $\vec{c}\left\{\begin{array}{l}\text { 方向 }: \vec{c} \perp \vec{a}, \vec{c} \perp \vec{b} \text { 且符合右手规则 } \\ \text { 模 ：}|\vec{c}|=|\vec{a}||\vec{b}| \sin \theta\end{array}\right.$
称 $\vec{c}$ 为向量 $\vec{a}$ 与 $\vec{b}$ 的向量积，记作
$\vec{c}=\vec{a} \times \vec{b}$（叉积）
引例中的力矩 $\vec{M}=\overrightarrow{O P} \times \vec{F}$

思考：右图三角形面积

$$
S=\frac{1}{2}|\vec{a} \times \vec{b}|
$$
