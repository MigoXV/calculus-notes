## 空间曲线小结与思考题

空间曲线的一般方程、参数方程．

$$
\left\{\begin{array} { l } 
{ F ( x , y , z ) = 0 } \\
{ G ( x , y , z ) = 0 }
\end{array} \quad \left\{\begin{array}{l}
x=x(t) \\
y=y(t) \\
z=z(t)
\end{array}\right.\right.
$$

空间曲线在坐标面上的投影．

$$
\left\{\begin{array} { l } 
{ H ( x , y ) = 0 } \\
{ z = 0 }
\end{array} \left\{\begin{array} { l } 
{ R ( y , z ) = 0 } \\
{ x = 0 }
\end{array} \left\{\begin{array}{l}
T(x, z)=0 \\
y=0
\end{array}\right.\right.\right.
$$

## 思考题解答

交线方程为 $\left\{\begin{array}{l}2 y^{2}+x^{2}=z \\ 2-x^{2}=z\end{array}\right.$ ，

消去 $z$ ，得投影柱面 $x^{2}+y^{2}=1$ ，
在 $x o y$ 面上的投影为 $\left\{\begin{array}{l}x^{2}+y^{2}=1 \\ z=0\end{array}\right.$ ．

## 练习题答案

一、1．$\left\{\begin{array}{l}y^{2}=\frac{10}{9} z \\ x=0\end{array}\right.$ ；
2． $3 y^{2}-z^{2}=16,3 x^{2}+2 z^{2}=16$ ；
3．$\left\{\begin{array}{l}x^{2}+4 z^{2}-2 x-3=0 \\ y=0\end{array}\right.$ ；
4．两直线的交点，两平面的交线；
5．椭圆与其一切线的交点，椭圆柱面 $\frac{x^{2}}{4}+\frac{y^{2}}{9}=1$ 与其切平面 $y=3$ 的交线；
6．$x^{2}+y^{2} \leq 4, y^{2} \leq z<4, x^{2} \leq z \leq 4$ ．

四、 $\left\{\begin{array}{l}x^{2}+y^{2}=a^{2} \\ z=0\end{array},\left\{\begin{array}{l}z=b \arcsin \frac{y}{a} \\ x=0\end{array},\left\{\begin{array}{l}z=b \arccos \frac{x}{a} \\ y=0\end{array}\right.\right.\right.$ ．
五、 $x^{2}+y^{2} \leq a x ; \quad z^{2}+a x \leq a^{2}, x \geq 0, z \geq 0$ 。
