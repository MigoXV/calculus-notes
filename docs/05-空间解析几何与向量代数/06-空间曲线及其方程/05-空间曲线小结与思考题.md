## 空间曲线小结与思考题

空间曲线的一般方程、参数方程．

$$
\left\{\begin{array} { l } 
{ F ( x , y , z ) = 0 } \\
{ G ( x , y , z ) = 0 }
\end{array} \quad \left\{\begin{array}{l}
x=x(t) \\
y=y(t) \\
z=z(t)
\end{array}\right.\right.
$$

空间曲线在坐标面上的投影．

$$
\left\{\begin{array} { l } 
{ H ( x , y ) = 0 } \\
{ z = 0 }
\end{array} \left\{\begin{array} { l } 
{ R ( y , z ) = 0 } \\
{ x = 0 }
\end{array} \left\{\begin{array}{l}
T(x, z)=0 \\
y=0
\end{array}\right.\right.\right.
$$
