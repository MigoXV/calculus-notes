一、应用微分方程解决实际问题的一般步骤
1、根据问题的实际背景，利用数学和有关学科知识，建立微分方程，确定定解条件；

2、根据方程的类型，用适当的方法求出方程的通解；

3、对所得结果进行具体分析，解释其实际意义。如果它与实际相差太远，则就应该修改模型，重新求解。

1．建立数学模型 —列微分方程问题

$$
\begin{array}{ll}
\text { 建立微分方程 (共性 ) } & \left\{\begin{array}{l}
\text { 利用物理规律 } \\
\text { 利用几何关系 }
\end{array}\right. \\
\text { 确定定解条件 (个性 ) } & \left\{\begin{array}{l}
\text { 初始条件 } \\
\text { 边界条件 } \\
\text { 可能还有衔接条件 }
\end{array}\right.
\end{array}
$$
