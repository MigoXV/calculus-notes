## 分类4：线性方程与非线性方程

若一个方程对未知函数及其导数的全体而言是一次的，且系数只与自变量有关（与未知函数及其导数无关），则称该方程为线性方程，否则，称之为非线性方程。

$$
\begin{array}{ll}
\frac{d x}{d t}=x^{2} & \text { 一阶 非线性 } \\
\frac{d^{2} y}{d x^{2}}+b \frac{d y}{d x}+c y=\sin x & \text { 二阶 线性 } \\
\left(\frac{d x}{d t}\right)^{2}+x^{2}=t^{5} & \text { 一阶 非线性 }
\end{array}
$$
