## 分类5：齐次方程、非齐次方程

在方程中，不含未知函数及其导数的项，称为自由项．自由项为零的方程，称为齐次方程．

自由项不为零的方程，称为非齐次方程．

$$
\begin{aligned}
& \frac{d x}{d t}=x^{2} \quad \text { 一阶齐次非线性方程 } \\
& \frac{d^{2} y}{d x^{2}}+b \frac{d y}{d x}+c y=\sin x \quad \text { 二阶非齐次线性方程 } \\
& \left(\frac{d x}{d t}\right)^{2}+x^{2}=t^{5} \quad \text { 一阶非齐次非线性方程 }
\end{aligned}
$$
