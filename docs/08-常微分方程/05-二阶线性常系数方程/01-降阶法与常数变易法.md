## 降阶法与常数变易法

1．齐次线性方程求线性无关特解————降阶法设 $y_{1}$ 是齐次方程

$$
\begin{equation*}
y^{\prime \prime}+P(x) y^{\prime}+Q(x) y=0 \tag{1}
\end{equation*}
$$

的一个非零特解，
令 $y_{2}=u(x) y_{1}$ 代入（1）式，得
$y_{1} u^{\prime \prime}+\left(2 y_{1}^{\prime}+P(x) y_{1}\right) u^{\prime}+\left(y_{1}^{\prime \prime}+P(x) y_{1}^{\prime}+Q(x) y_{1}\right) u=0$,
即 $y_{1} u^{\prime \prime}+\left(2 y_{1}^{\prime}+P(x) y_{1}\right) u^{\prime}=0$ ，令 $v=u^{\prime}$ ，

则有 $y_{1} v^{\prime}+\left(2 y_{1}^{\prime}+P(x) y_{1}\right) v=0$ ，
$y_{1} v^{\prime}+\left(2 y_{1}^{\prime}+P(x) y_{1}\right) v=0 \quad v$ 的一阶方程
降阶法
解得 $v=\frac{1}{y_{1}^{2}} e^{-\int P(x) d x}, \quad \therefore u=\int \frac{1}{y_{1}^{2}} e^{-\int P(x) d x} d x$
$\therefore y_{2}=y_{1} \int \frac{1}{y_{1}^{2}} e^{-\int P(x) d x} d x$ ，
齐次方程通解为
刘维尔公式
$y=C_{1} y_{1}+C_{2} y_{1} \int \frac{1}{y_{1}^{2}} e^{-\int P(x) d x} d x$.

2．非齐次线性方程通解求法———常数变易法
非齐次方程 $y^{\prime \prime}+P(x) y^{\prime}+Q(x) y=f(x)$

设对应齐次方程通解为 $\quad y=C_{1} y_{1}+C_{2} y_{2}$

设非齐次方程通解为 $\quad y=c_{1}(x) y_{1}+c_{2}(x) y_{2}$

$$
y^{\prime}=c_{1}^{\prime}(x) y_{1}+c_{2}^{\prime}(x) y_{2}+c_{1}(x) y_{1}^{\prime}+c_{2}(x) y_{2}^{\prime}
$$

设 $c_{1}^{\prime}(x) y_{1}+c_{2}^{\prime}(x) y_{2}=0$
$y^{\prime \prime}=c_{1}^{\prime}(x) y_{1}^{\prime}+c_{2}^{\prime}(x) y_{2}^{\prime}+c_{1}(x) y_{1}^{\prime \prime}+c_{2}(x) y_{2}^{\prime \prime}$

将 $y, y^{\prime}, y^{\prime \prime}$ 代入方程（2），得

$$
\begin{aligned}
& c_{1}^{\prime}(x) y_{1}^{\prime}+c_{2}^{\prime}(x) y_{2}^{\prime}+c_{1}(x)\left(y_{1}^{\prime \prime}+P(x) y_{1}^{\prime}+Q(x) y_{1}\right) \\
& \quad+c_{2}(x)\left(y_{2}^{\prime \prime}+P(x) y_{2}^{\prime}+Q(x) y_{2}\right)=f(x)
\end{aligned}
$$

$$
\begin{equation*}
c_{1}^{\prime}(x) y_{1}^{\prime}+c_{2}^{\prime}(x) y_{2}^{\prime}=f(x) \tag{5}
\end{equation*}
$$

（4），（5）联立方程组 $\quad\left\{\begin{array}{l}c_{1}^{\prime}(x) y_{1}+c_{2}^{\prime}(x) y_{2}=0 \\ c_{1}^{\prime}(x) y_{1}^{\prime}+c_{2}^{\prime}(x) y_{2}^{\prime}=f(x)\end{array}\right.$
∵ 系数行列式 $w(x)=\left|\begin{array}{ll}y_{1} & y_{2} \\ y_{1}^{\prime} & y_{2}^{\prime}\end{array}\right| \neq 0$,

$$
\therefore c_{1}^{\prime}(x)=-\frac{y_{2} f(x)}{w(x)}, \quad c_{2}^{\prime}(x)=\frac{y_{1} f(x)}{w(x)},
$$

积分可得 $\quad c_{1}(x)=C_{1}+\int-\frac{y_{2} f(x)}{w(x)} d x$ ，

$$
c_{2}(x)=C_{2}+\int \frac{y_{1} f(x)}{w(x)} d x
$$

非齐次方程通解为

$$
y=C_{1} y_{1}+C_{2} y_{2}-y_{1} \int \frac{y_{2} f(x)}{w(x)} d x+y_{2} \int \frac{y_{1} f(x)}{w(x)} d x
$$

例1 求方程 $y^{\prime \prime}+\frac{x}{1-x} y^{\prime}-\frac{1}{1-x} y=x-1$ 的通解。

例2 求方程 $x^{2} y^{\prime \prime}-(x+2)\left(x y^{\prime}-y\right)=x^{4}$ 的通解．

例1 求方程 $y^{\prime \prime}+\frac{x}{1-x} y^{\prime}-\frac{1}{1-x} y=x-1$ 的通解。
解 $\because 1+\frac{x}{1-x}-\frac{1}{1-x}=0$ ，
对应齐方一特解为 $\boldsymbol{y}_{1}=\boldsymbol{e}^{x}$ ，由刘维尔公式

$$
y_{2}=e^{x} \int \frac{1}{e^{2 x}} e^{-\int \frac{x}{1-x} d x} d x=x
$$

对应齐方通解为 $\quad Y=C_{1} x+C_{2} e^{x}$ ．

设原方程的通解为 $y=c_{1}(x) x+c_{2}(x) e^{x}$ ， $c_{1}^{\prime}(x), c_{2}^{\prime}(x)$ 应满足方程组

$$
\begin{aligned}
& \left\{\begin{array} { l } 
{ x c _ { 1 } ^ { \prime } ( x ) + e ^ { x } c _ { 2 } ^ { \prime } ( x ) = 0 } \\
{ c _ { 1 } ^ { \prime } ( x ) + e ^ { x } c _ { 2 } ^ { \prime } ( x ) = x - 1 }
\end{array} \text { 解得 } \quad \left\{\begin{array}{l}
c_{1}^{\prime}(x)=-1 \\
c_{2}^{\prime}(x)=x e^{-x}
\end{array}\right.\right. \\
& c_{1}(x)=-x+C_{1}, \quad c_{2}(x)=-x e^{-x}-e^{-x}+C_{2}
\end{aligned}
$$

原方程的通解为 $y=C_{1} x+C_{2} e^{x}-x^{2}-x-1$ ．

例2 求方程 $x^{2} y^{\prime \prime}-(x+2)\left(x y^{\prime}-y\right)=x^{4}$ 的通解．
解：对应齐次方程为 $x^{2} y^{\prime \prime}-(x+2)\left(x y^{\prime}-y\right)=0$
由观察可知它有特解：$y_{1}=x$ ，
令 $y=x u(x)$ ，代入非齐次方程后化简得

$$
u^{\prime \prime}-u^{\prime}=x
$$

解上述可降阶微分方程，可得通解：

$$
u=C_{1}+C_{2} \mathrm{e}^{x}-\left(\frac{1}{2} x^{2}+x\right)
$$

故原方程通解为

$$
y=x u=C_{1} x+C_{2} x \mathrm{e}^{x}-\left(\frac{1}{2} x^{3}+x^{2}\right)
$$

补充内容

$$
y^{\prime \prime}+P(x) y^{\prime}+Q(x) y=0
$$

可观察出一个特解
（1）若 $P(x)+x Q(x)=0, \quad$ 特解 $y=x$ ；
（2）若 $1+P(x)+Q(x)=0$ ，特解 $y=e^{x}$ ；
（3）若 $1-P(x)+Q(x)=0$ ，特解 $y=e^{-x}$ ．
