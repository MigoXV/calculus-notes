## 2）有阻尼自由振动情况

方程：$\quad \frac{\mathrm{d}^{2} x}{\mathrm{~d} t^{2}}+2 n \frac{\mathrm{~d} x}{\mathrm{~d} t}+k^{2} x=0$
特征方程：$\quad r^{2}+2 n r+k^{2}=0$
特征根：$\quad r_{1,2}=-n \pm \sqrt{n^{2}-k^{2}}$
这时需分如下三种情况进行讨论：
小阻尼： $\boldsymbol{n}<\boldsymbol{k} \Longrightarrow x=\mathrm{e}^{-n t}\left(C_{1} \cos \omega t+C_{2} \sin \omega t\right)$

$$
\left(\omega=\sqrt{k^{2}-n^{2}}\right) \text { 解的特征 }
$$

大阻尼： $\boldsymbol{n > k} \Longrightarrow x=C_{1} \mathrm{e}^{r_{1} t}+C_{2} \mathrm{e}^{r_{2} t}$ $\_\_\_\_$
的特征
临界阻尼： $\boldsymbol{n = k} \Longrightarrow x=\left(C_{1}+C_{2} t\right) \mathrm{e}^{-n t}$ $\_\_\_\_$军的特征
