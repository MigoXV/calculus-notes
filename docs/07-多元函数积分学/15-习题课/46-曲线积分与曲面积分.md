## 1.曲线积分与曲面积分

|  | 曲 线 积 分 |  |
| :--- | :--- | :--- |
|  | 对弧长的曲线积分 | 对坐标的曲线积分 |
| 定义 | $\int_{L} f(x, y) d s=\lim _{\lambda \rightarrow 0} \sum_{i=1}^{n} f\left(\xi_{i}, \eta_{i}\right) \Delta s_{i}$ | $\int_{L} P(x, y) d x+Q(x, y) d y$ $=\lim _{\lambda \rightarrow 0} \sum_{i=1}^{n}\left[P\left(\xi_{i}, \eta_{i}\right) \Delta x_{i}+Q\left(\xi_{i}, \eta_{i}\right) \Delta y_{i}\right]$ |
| 自系 | $\int_{L} P d x+Q d y=\int_{L}(P \cos \alpha+Q \cos \beta) d s$ |  |
| 计 | $\begin{aligned} & \int_{L} f(x, y) d s \\ & =\int_{\alpha}^{\beta} f[\varphi, \Psi] \sqrt{\varphi^{\prime 2}+\psi^{\prime 2}} d t \\ & \text { 一代二换三定限 } \quad(\alpha<\beta) \end{aligned}$ | $\begin{aligned} & \int_{L} P d x+Q d y \\ & =\int_{\alpha}^{\beta}\left[P(\varphi, \psi) \varphi^{\prime}+Q(\varphi, \psi) \psi^{\prime}\right] d t \\ & \text { 一代二换三定向 (限) } \end{aligned}$ |

|  | 与路径无关的四个等价命题 |
| :--- | :--- |
| 条 <br> 件 | 在单连通开区域 $\boldsymbol{D}$ 上 $\boldsymbol{P}(\boldsymbol{x}, \boldsymbol{y}), \boldsymbol{Q}(\boldsymbol{x}, \boldsymbol{y})$ 具有 <br> 续的一阶偏导数，则以下四个命题成立． |
| 等 | $(1) \quad$ 在 $\boldsymbol{D}$ 内 $\int_{L} \boldsymbol{P} d x+\boldsymbol{Q} d \boldsymbol{y}$ 与路径无关 |
| 价 | $(2) \quad \oint_{C} \boldsymbol{P} d x+\boldsymbol{Q} d \boldsymbol{y}=\mathbf{0}$, 闭曲线 $C \subset \boldsymbol{D}$ |
| 命 | $(3) \quad$ 在 $\boldsymbol{D}$ 内存在 $U(\boldsymbol{x}, y)$ 使 $d u=\boldsymbol{P} d x+\boldsymbol{Q} d \boldsymbol{y}$ |
| 题 | $(4) \quad$ 在 $\boldsymbol{D}$ 内，$\frac{\partial \boldsymbol{P}}{\partial y}=\frac{\partial \boldsymbol{Q}}{\partial x}$ |

|  | 曲 面 积 分 |  |
| :--- | :--- | :--- |
|  | 对面积的曲面积分 | 对坐标的曲面积分 |
| 定 义 | $\iint_{\Sigma} f(x, y, z) d s=\lim _{\lambda \rightarrow \sum_{i=1}^{n}} \sum_{i=1}^{n} f\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta s_{i}$ | $\iint_{\Sigma} R(x, y, z) d x d y=\lim _{\lambda \rightarrow \rightarrow} \sum_{i=1}^{n} R\left(\xi_{i}, \eta_{i}, \zeta_{i}\right)\left(\Delta S_{i}\right)_{N_{i}}$ |
| 联 | $\iint_{\Sigma} P d y d z+Q d z d x+R d x d y=\iint_{\Sigma}(P \cos \alpha+Q \cos \beta+R \cos \gamma) d S$ |  |
| 计 | $\begin{aligned} & \iint_{\Sigma} f(x, y, z) d s \\ &= \iint_{D_{v}} f(x, y, z(x, y)) \mid \sqrt{1+z_{x}^{2}+z_{y}^{2}} d x d y \\ & \text { 一代,二换,三投四定限(与侧无关) } \end{aligned}$ | $\begin{aligned} & \iint_{\Sigma} R(x, y, z) d x d y \\ & = \pm \iint_{D_{y}} R(x, y, z(x, y)) d x d y \\ & \text { 一代,二投,三定向 (与侧有关) } \end{aligned}$ |
