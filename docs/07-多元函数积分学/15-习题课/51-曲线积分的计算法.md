## 二、曲线积分的计算法

1．基本方法
曲线积分 $\left\{\begin{array}{l}\text { 第一类（对弧长）} \\ \text { 第二类（对坐标）}\end{array}\right\} \xrightarrow[\text { 转化 }]{ }$ 定积分
（1）统一积分变量
第一类：$\left\{\begin{array}{l}\text { 用参数方程 } \\ \text { 用直角坐标方程 } \\ \text { 用极坐标方程 }\end{array} \quad\right.$ 第二类：$\left\{\begin{array}{l}\text { 用参数方程 } \\ \text { 用直角坐标方程 }\end{array}\right.$
（2）确定积分上下限 $\left\{\begin{array}{l}\text { 第一类：下小上大 } \\ \text { 第二类：下始上终 }\end{array}\right.$
