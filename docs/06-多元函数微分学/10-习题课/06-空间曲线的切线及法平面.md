## 1.空间曲线的切线及法平面

（1）设空间曲线：
$x=x(t), y=y(t), z=z(t) \quad$（ $t$ 为参数）
$M_{0}\left(x_{0}, y_{0}, z_{0}\right)$ 是曲线上一点，其相应的参数为 $t_{0}$ ，
则曲线在点 $M_{0}$ 处切向量为

$$
\vec{T}=\left\{x^{\prime}\left(t_{0}\right), y^{\prime}\left(t_{0}\right) z^{\prime}\left(t_{0}\right)\right\},
$$

曲线在点 $M_{0}$ 处的切线方程为

$$
\frac{x-x_{0}}{x^{\prime}\left(t_{0}\right)}=\frac{y-y_{0}}{y^{\prime}\left(t_{0}\right)}=\frac{z-z_{0}}{z^{\prime}\left(t_{0}\right)}
$$

曲线在点 $M_{0}$ 处的法平面方程为

$$
x^{\prime}\left(t_{0}\right)\left(x-x_{0}\right)+y^{\prime}\left(t_{0}\right)\left(y-y_{0}\right)+z^{\prime}\left(t_{0}\right)\left(z-z_{0}\right)=0
$$

若曲线的方程表示为

$$
\left\{\begin{array} { l } 
{ y = y ( x ) } \\
{ z = z ( x ) }
\end{array} \Leftarrow \left\{\begin{array}{l}
F(x, y, z)=0 \\
G(x, y, z)=0
\end{array}\right.\right.
$$

则在点 $M_{0}$ 处切向量为

$$
\vec{T}=\left\{1, y^{\prime}\left(x_{0}\right) z^{\prime}\left(x_{0}\right)\right\}
$$
