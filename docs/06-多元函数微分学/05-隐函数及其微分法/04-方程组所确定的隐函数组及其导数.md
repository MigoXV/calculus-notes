## 二、方程组所确定的隐函数组及其导数

在此举例说明求偏导的方法，方程组确定的隐函数一般有以下几种情形：

1．$\left\{\begin{array}{l}F(x, y, z)=0 \\ G(x, y, z)=0\end{array}\right.$ 确定了两个一元函数。
2．$\left\{\begin{array}{l}F(x, y, u, v)=0 \\ G(x, y, u, v)=0\end{array}\right.$ 确定了两个二元函数。
3． $\begin{cases}x=x(u, v) & \\ y=y(u, v) & \text { 确定了一个以 } u, v \text { 为中间变量 } \\ z=z(u, v) & x, y \text { 为自变量的二元函数 } z=z(x, y) .\end{cases}$
