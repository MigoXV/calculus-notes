## 一、多元函数的极值

1．二元函数极值的定义
设 $z=f(x, y)$ 在 $U\left(P_{0}\left(x_{0}, y_{0}\right), \delta\right)$ 内有定义，对于一切异于 $\boldsymbol{P}_{\mathbf{0}}$ 的点 $\boldsymbol{P}(\boldsymbol{x}, \boldsymbol{y})$ ，若都适合不等式

$$
f(x, y)<f\left(x_{0}, y_{0}\right)\left(\text { 或 } f(x, y)>f\left(x_{0}, y_{0}\right)\right)
$$

则称 $z=f(x, y)$ 在 $P_{0}\left(x_{0}, y_{0}\right)$ 有极大值 或极小值 $f\left(x_{0}, y_{0}\right)$ ．极大值与极小值统称为极值。 $P_{0}\left(x_{0}, y_{0}\right)$ 为极值点。

若引进点函数，则当 $f(P)<f\left(P_{0}\right)$ 时，$f\left(P_{0}\right)$ 为极大值；

$$
\text { 当 } f(P)>f\left(P_{0}\right) \text { 时, } f\left(P_{0}\right) \text { 为极小值. }
$$

函数 $z=3 x^{2}+4 y^{2}$在 $(0,0)$ 处有极小值．

函数 $z=-\sqrt{x^{2}+y^{2}}$在 $(0,0)$ 处有极大值．

函数 $z=x y$在 $(0,0)$ 处无极值．
