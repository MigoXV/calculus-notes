## 我们也可以反向操作，由等高线图推敲出三维的函数图形．

例题 2 图 8.7 中的等高线代表一个曲面，请问是啥曲面？

图8.7 某个曲重的等高线固

解：我们从图上的等高线看到，这个曲面与 $x 0 y$ 平面上方的那些水平平面 （即 $z>0$ 者）．相交于一些不经过 $y 0 z$ 平面的双曲线，而且与 $x 0 y$ 平面下方的水平平面（即 $z<0$ 者），相交于一些不经过 $x 0 z$ 平面的双曲线．至于它与 $x 0 y$平面本身的交线，则为两条直交的对角线．具有这样特性的曲面，就是每一位牛仔心中最怗念的曲面一马鞍．最简单的鞍面方程式就是 $z=x^{2}-y^{2}$ ，下图显示的就是它的圆形．
