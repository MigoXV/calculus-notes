# 高等数学笔记

这是一个按课程章节整理的高等数学 Markdown 笔记仓库，内容覆盖极限与连续、一元微积分、无穷级数、空间解析几何、多元微积分和常微分方程等主题。仓库中的正式笔记主要放在编号目录 `01` 到 `08` 下，适合按课程顺序阅读，也可以作为复习时的公式、例题和方法索引。

## 目录结构

```text
.
├── 01-极限与连续
├── 02-一元函数微分学
├── 03-一元函数积分学
├── 04-无穷级数
├── 05-空间解析几何与向量代数
├── 06-多元函数微分学
├── 07-多元函数积分学
├── 08-常微分方程
├── data-bin
└── tmp-workspace
```

- `01-极限与连续`：函数、数列极限、函数极限、无穷小与连续性。
- `02-一元函数微分学`：导数、求导法则、中值定理、泰勒公式、罗必塔法则和函数性态。
- `03-一元函数积分学`：不定积分、换元积分法、定积分、微积分基本定理、几何应用和广义积分。
- `04-无穷级数`：常数项级数、函数项级数、幂级数和 Fourier 级数。
- `05-空间解析几何与向量代数`：空间坐标系、向量运算、平面与空间直线、曲面和空间曲线。
- `06-多元函数微分学`：多元函数、偏导数、全微分、复合函数、隐函数、方向导数、梯度和极值。
- `07-多元函数积分学`：重积分、曲线积分、曲面积分、Green 公式、Gauss 公式和 Stokes 公式。
- `08-常微分方程`：一阶方程、可降阶方程、二阶线性方程、常系数方程和简单应用。

`data-bin` 与 `tmp-workspace` 用于存放整理过程中的中间材料或临时工作内容，阅读正式笔记时通常不需要进入。

## 阅读建议

建议从一级编号目录按顺序阅读。每个目录下的文件名通常以两位编号开头，表示该章内部的学习顺序；部分章节还包含“习题课”“小结”“练习题答案”等复习材料。

笔记中包含 LaTeX 公式和少量 Mermaid 图表。推荐使用支持数学公式渲染的 Markdown 编辑器或笔记工具阅读，例如 VS Code 搭配 Markdown Preview Enhanced，或其他支持 MathJax/KaTeX 与 Mermaid 的阅读环境。

## 构建网站

本仓库已配置 MkDocs，可将现有章节笔记直接构建为静态网站。依赖安装在项目虚拟环境中：

```bash
.venv/bin/python -m pip install -r requirements-mkdocs.txt
```

本地预览：

```bash
.venv/bin/mkdocs serve
```

生成静态站点：

```bash
.venv/bin/mkdocs build
```

站点会按 `01` 到 `08` 的正式笔记目录自动生成导航，`data-bin`、`tmp-workspace`、`.venv` 和 `site` 等目录不会进入网站内容。

## 维护约定

- 新增正式笔记时，优先放入对应的编号章节目录。
- 文件名尽量保持 `两位编号-主题.md` 的形式，便于排序和检索。
- 公式统一使用 Markdown 中的 LaTeX 写法，行间公式使用 `$$ ... $$`。
- 图示流程可继续使用 Mermaid 代码块。
- README 与面向读者的说明文档使用中文编写。
